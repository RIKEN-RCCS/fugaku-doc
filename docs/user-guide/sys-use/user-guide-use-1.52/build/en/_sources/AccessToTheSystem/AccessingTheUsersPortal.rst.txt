Accessing steps to the Fugaku website
======================================


This indicates how to access to the Fugaku website.

1. | Open the Web browser and acces to the following URL.
   | https://www.fugaku.r-ccs.riken.jp/en


 .. note::
 
   - The Fugaku website has been tested on Mozilla Firefox and Google Chrome. If you are using other Web browsers and you have problems with the operation, please use a Web browser which operation has been confirmed. In addition, when using Microsoft Internet Explorer, it is confirmed that abnormal termination occurs with :ref:`RegisteringAPublicKey`.
   - To prevent vulnerabilities, the Fugaku website prohibits old SSL connections and accepts only TLS 1.2 and TLS 1.3 connections.Depending on the settings of your Web browser, you may not be able to connect, so please change the settings appropriately to use TLS 1.2 or later as follows.

       [Setting change direcion on Firefox]
   
       1. Enter  \ :command:`about:config`\  on the address bar and press an enter key.
       2. Search with  ``security.tls.version``\.
       3. Confirm if  :guilabel:`[security.tls.version.max]`\  is 4 (Up to TLS 1.3 is available)
       4. If smaller than 4, set 4.


2. Once the client certificate selection diarogue is shown, select the using lolal account's client certificate.

   - The example of Firefox diarogue
   
    .. figure:: ./img/AccessingTheUserPortal_02.png
       :align: center

   - The example of Chrome diarogue
   
    .. figure:: ./img/AccessingTheUserPortal_07.png
       :align: center

#. In the password input dialog, enter the password of the private key registered when the client certificate was installed. If you're using macOS Keychain, enter the Keychain password.


   - The example of Firefox diarogue
   
    .. figure:: ./img/AccessingTheUserPortal_04.png
       :scale: 80
       :align: center

   - The example of Keychain (macOS)
   
    Enter the keychain password (usually the same as your computer login password).

    .. figure:: ./img/AccessingTheUserPortal_06.png
       :scale: 70
       :align: center

    .. note::

       If you are repeatedly prompted to enter a password for the keychain when using Chrome on your Mac, click "Always Allow" in the password input dialog.

#. If the client certificate is successfully authenticated, the following screen will be displayed.

    .. figure:: ./img/AccessingTheUserPortal_05.png
       :scale: 50
       :align: center

#. Logout

   The Fugaku website does not have logout function.
   Please close your Web browser to stop using of the Fugaku website completely.   

   If you have multiple client certificates (Accounts), you can switch to another account by using either of the methods described below. 

   - Close your Web browser and access the Fugaku website again to select another client certificate.
   - Access the Fugaku website using another Web browser (Firefox or Google Chrome) to select another client certificate.
