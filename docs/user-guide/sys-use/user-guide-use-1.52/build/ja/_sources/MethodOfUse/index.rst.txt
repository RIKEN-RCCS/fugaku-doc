=========
利用方法
=========

「富岳」の利用方法を示します。

概要
==========================


「富岳」の利用を開始するためには「スタートアップガイド」に記載した以下の手続きが必要です。

- :ref:`クライアント証明書のインストール <InstallingClientCertificate>`
- :ref:`SSH公開鍵の登録 <AccessToTheSystemLogin>`


プログラムの作成、コンパイルからジョブの実行までの一連の流れを以下に示します。

..
  .. figure:: ./img/OverviewOfUsage_01.png
     :scale: 80
     :align: center
..

1. ログインノードへのログイン

  .. code-block:: console
  
   [terminal] $ ssh username@login.fugaku.r-ccs.riken.jp

2. プログラム作成
 
  .. code-block:: console
  
   [_LNlogin] $ emacs sample.f08


3. コンパイル
 
  .. code-block:: console
  
   [_LNlogin] $ frtpx -Kfast,parallel -o sample sample.f08

4. ジョブスクリプト作成
 
  .. code-block:: console
  
   [_LNlogin] $ emacs sample.sh

5. ジョブ実行
 
  .. code-block:: console
  
   [_LNlogin] $ pjsub sample.sh

6. ジョブ状態表示
 
  .. code-block:: console
  
   [_LNlogin] $ pjstat

7. 結果確認

  .. code-block:: console
  
   [_LNlogin] $ less ./sample.sh.jobid.out
   [_LNlogin] $ less ./sample.sh.jobid.err
   [_LNlogin] $ less ./sample.sh.jobid.stats

マニュアル
==================

| 以下のマニュアルを富岳ウェブサイトで提供しています。
| 富岳ウェブサイトのメニューから「ドキュメント」->「\ `利用手引書 <https://www.fugaku.r-ccs.riken.jp/docs/guides_r01>`_\ 」あるいは「\ `マニュアル <https://www.fugaku.r-ccs.riken.jp/docs/manuals_r01>`_\ 」を参照ください。
| より詳細な機能を利用したい場合には関連するマニュアルを参照してください。

運用ソフト関連マニュアルおよびファイルシステム関連マニュアルは、機種ごとの固有の仕様を記載した箇所があります。
「富岳」は「FXサーバ」または［FX］と書かれた箇所を参照ください。

言語関連マニュアルには「/製品インストールパス/」と書かれた箇所があります。
「/製品インストールパス/」は版数により異なります。
言語環境を設定するmodulefileで環境変数FJSVXTCLANGAに「/製品インストールパス/」を設定しています。
