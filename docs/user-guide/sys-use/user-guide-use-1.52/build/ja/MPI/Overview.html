6.1. 概要¶

MPIプログラムを計算ノード上で実行するには、mpiexecコマンドを使用します。

mpiexecコマンドは、ジョブスクリプト内に記述する必要があります。
MPIプログラムを実行するジョブの投入は、pjsubコマンドで--mpiオプションに以下のパラメーターを指定して行います。

  • プロセスの形状

  • 生成するプロセス数

  • 生成するプロセスのランクの割り当てルール
