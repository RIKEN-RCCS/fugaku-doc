6.7. ジョブ形状（トーラス）¶

pjsub -L nodeで指定された形状（1次元／2次元／3次元）、計算ノード数に応じて、ジョブに資源が割り当てられます。

割り当てられた資源は、各軸単位にジョブ内でトーラスが構成されます。
