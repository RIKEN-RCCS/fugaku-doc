2. 利用方法¶

「富岳」の利用方法を示します。

2.1. 概要¶

「富岳」の利用を開始するためには「スタートアップガイド」に記載した以下の手続きが必要です。

  • クライアント証明書のインストール

  • SSH公開鍵の登録

プログラムの作成、コンパイルからジョブの実行までの一連の流れを以下に示します。

  1. ログインノードへのログイン

[terminal] $ ssh username@login.fugaku.r-ccs.riken.jp

  1. プログラム作成

[_LNlogin] $ emacs sample.f08

  1. コンパイル

[_LNlogin] $ frtpx -Kfast,parallel -o sample sample.f08

  1. ジョブスクリプト作成

[_LNlogin] $ emacs sample.sh

  1. ジョブ実行

[_LNlogin] $ pjsub sample.sh

  1. ジョブ状態表示

[_LNlogin] $ pjstat

  1. 結果確認

[_LNlogin] $ less ./sample.sh.jobid.out
[_LNlogin] $ less ./sample.sh.jobid.err
[_LNlogin] $ less ./sample.sh.jobid.stats

2.2. マニュアル¶

以下のマニュアルを富岳ウェブサイトで提供しています。
富岳ウェブサイトのメニューから「ドキュメント」->「利用手引書」あるいは「マニュアル」を参照ください。
より詳細な機能を利用したい場合には関連するマニュアルを参照してください。

運用ソフト関連マニュアルおよびファイルシステム関連マニュアルは、機種ごとの固有の仕様を記載した箇所があります。 「富岳」は「FXサーバ」または［FX］と書かれた箇所を参照ください。

言語関連マニュアルには「/製品インストールパス/」と書かれた箇所があります。 「/製品インストールパス/」は版数により異なります。 言語環境を設定するmodulefileで環境変数FJSVXTCLANGAに「/製品インストールパス/」を設定しています。
