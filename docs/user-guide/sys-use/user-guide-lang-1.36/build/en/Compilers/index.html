2. Compiler type¶

Multiple compilers are available with the Supercomputer Fugaku. For information on using the compiler, see:.

  • Compilers that generate binaries for compute nodes

  • Compilers that generate binaries for login nodes

Programs that run on compute nodes are compiled using Compilers that generate binaries for compute nodes.

Use the cross-compilers Fujitsu compiler, LLVM, and GCC to generate binaries that run on compute nodes on login nodes.
