Sequential execution
====================

The following is an example of compiling a single node job (sequential) C program.


1. | Prepare source program.
   | Prepare sample program as  \ :file:`/home/system/sample/C/single/trad_sample.c`\.
   | :download:`(Download sample code) <./src/trad_sample.c>`


.. code-block:: c
   :linenos:

   #include <stdio.h>
   #include <math.h>
   #include <sys/time.h>
   
   #define N  5888
   
   double gettimeofday_sec(){
       struct timeval tv;
       gettimeofday(&tv, NULL);
       return tv.tv_sec + (double)tv.tv_usec*1e-6;
   }
   
   int main(){
       double a[N][N+1];
       double b[N][N+1];
       double c[N][N+1];
       int i, j, k;
       double t1,t2;
   
   //
   //  print message
   //
       printf ("\n");
       printf ("%d, %d\n", N, N);
   //
   //  initialize matrix
   //
       for (j = 0; j < N; j++) {
          for (i = 0; i < N; i++) {
             a[j][i] = 0.1;
             b[j][i] = 0.3;
             c[j][i] = 0.0;
          }
       }
   
   //
   //  do matrix multiply
   //
       t1 = gettimeofday_sec();
       for (j = 0; j < N; j++) {
          for (i = 0; i < N; i++) {
             for (k = 0; k < N; k++) {
                c[i][j] = c[i][j] + a[i][k] * b[k][j];
             }
          }
       }
       t2 = gettimeofday_sec();
   
   //
   //  print result
   //
       printf("c[1][1] = %f\n", c[1][1]);
       printf("time : %f sec.\n", t2 - t1);
       return 0;
   }

3. | Prepare job script.
   | Prepare job script sample as  \ :file:`/home/system/sample/C/single/trad_job_single.sh`\.
   | :download:`(Download sample code) <./src/trad_job_single.sh>`

 .. code-block:: bash

  #!/bin/sh
  #PJM -L "node=1"
  #PJM -L "rscgrp=small"
  #PJM -L "elapse=10:00"
  #PJM -x PJM_LLIO_GFSCACHE=/vol000N
  #PJM -g groupname
  #PJM -s

  # execute job
  ./sample


4. Submit a job with  \ :program:`pjsub`\  command.

 .. code-block:: console
  
  [_LNlogin]$ pjsub trad_job_single.sh
  
  [INFO] PJM 0000 pjsub Job 115 submitted

5. | Chek execution result.
   | The standard output is output as  \ :file:`job name.job ID.out`\.

 .. code-block:: console
  
  [_LNlogin]$ cat trad_job_single.sh.115.out

  5888, 5888
  c[1][1] = 176.640000
  time : 1278.064804 sec.
