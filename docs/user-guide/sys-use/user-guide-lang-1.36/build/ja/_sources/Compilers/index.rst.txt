.. _CompilersType:

提供中のコンパイラ種別
======================

|post-k|\ では、複数のコンパイラを利用することができます。
コンパイラの使用方法は次のページを参照ください。

- :ref:`計算ノード向けバイナリを生成するコンパイラ <CompilerforCN>`
- :ref:`ログインノード向けバイナリを生成するコンパイラ <CompilerforLN>`

計算ノードで実行するプログラムは\ :ref:`計算ノード向けバイナリを生成するコンパイラ <CompilerforCN>`\ を使用してコンパイルします。

ログインノード上で計算ノードで実行するバイナリを生成する場合は、\ :ref:`富士通コンパイラ <FujitsuCompiler>`\ 、\ :ref:`LLVM <CompilerforCNLLVM>`\ 、\ :ref:`GCC <GCC_MPI>`\ のクロスコンパイラを利用します。
