概要
=========

プロファイラは、プログラムの実行性能を向上させるためのチューニングに有用な性能の計測、および、その情報を出力するツールです。
|post-k|\ では、"基本プロファイラ"、"詳細プロファイラ"、"CPU性能解析レポート"の3機能で構成しています。

 .. list-table::
   :header-rows: 1
   :widths: 3 2 4 3

   * - 機能分類
     - 測定コマンド
     - 区間指定関数
     - 後処理コマンド

   * - 基本プロファイラ
     - fipp -C
     - fipp_start/fipp_stop
     - fipppx -A

   * - 詳細プロファイラ
     - fapp -C
     - fapp_start/fapp_stop
     - fapppx -A

   * - CPU性能解析レポート
     - fapp -C
     - fapp_start/fapp_stop
     - fapppx -A -tcsv

 .. seealso::
   測定コマンドは、計算ノードで実行します。
   
   後処理コマンドは、ログインノードで実行します。
