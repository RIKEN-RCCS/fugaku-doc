3.1.8.3.7.2. Automatic parallel¶

The following is an example of compiling a single node job (automatic parallel) C program.

  1. Prepare sample program.
    Sample program is the same as the example “C sample program (sequential)”.
    (Download sample code)
 1#include <stdio.h>
 2#include <math.h>
 3#include <sys/time.h>
 4
 5#define N  5888
 6
 7double gettimeofday_sec(){
 8    struct timeval tv;
 9    gettimeofday(&tv, NULL);
10    return tv.tv_sec + (double)tv.tv_usec*1e-6;
11}
12
13int main(){
14    double a[N][N+1];
15    double b[N][N+1];
16    double c[N][N+1];
17    int i, j, k;
18    double t1,t2;
19
20//
21//  print message
22//
23    printf ("\n");
24    printf ("%d, %d\n", N, N);
25//
26//  initialize matrix
27//
28    for (j = 0; j < N; j++) {
29       for (i = 0; i < N; i++) {
30          a[j][i] = 0.1;
31          b[j][i] = 0.3;
32          c[j][i] = 0.0;
33       }
34    }
35
36//
37//  do matrix multiply
38//
39    t1 = gettimeofday_sec();
40    for (j = 0; j < N; j++) {
41       for (i = 0; i < N; i++) {
42          for (k = 0; k < N; k++) {
43             c[i][j] = c[i][j] + a[i][k] * b[k][j];
44          }
45       }
46    }
47    t2 = gettimeofday_sec();
48
49//
50//  print result
51//
52    printf("c[1][1] = %f\n", c[1][1]);
53    printf("time : %f sec.\n", t2 - t1);
54    return 0;
55}

Note

  • By specifying -Koptmsg=2, Messages are output regarding optimizations such as automatic parallelization, SIMD conversion, and loop unrolling. In the above example, jwd5001p-i and jwd6001s-i are applicable.

  • jwd5001p-i means that the loop has been parallelized. Auto-parallelized loops and variables are displayed.

  • jwd6001s-i means that the loop has been converted to SIMD. SIMD loops and variables are displayed.

  1. Prepare job script.
    Job script sample is prepared as /home/system/sample/C/parallel/trad_job_para.sh.
    (Download sample code)
#!/bin/sh
#PJM -L "node=1"
#PJM -L "rscgrp=small"
#PJM -L "elapse=10:00"
#PJM -x PJM_LLIO_GFSCACHE=/vol000N
#PJM -g groupname
#PJM -s

# execute job
export OMP_NUM_THREADS=12
./sample_parallel

  1. Submit a job with pjsub command.

[_LNlogin]$ pjsub trad_job_para.sh
 [INFO] PJM 0000 pjsub Job 116 submitted.
  1. Check execution result.
    The standard output is output as job name.job ID.out.
[_LNlogin]$ cat trad_job_para.sh.116.out

5888, 5888
c[1][1] = 176.640000
time : 24.958370 sec.
