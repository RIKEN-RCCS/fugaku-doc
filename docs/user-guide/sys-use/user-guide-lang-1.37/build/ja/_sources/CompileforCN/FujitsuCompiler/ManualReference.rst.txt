マニュアル
==================

| 以下のマニュアルを富岳ウェブサイトで提供しています。
| 富岳ウェブサイトのメニューから「ドキュメント」->「\ `マニュアル <https://www.fugaku.r-ccs.riken.jp/docs/manuals_r01#LANGMAN>`_\ 」を参照ください。
| より詳細な機能を利用したい場合には関連するマニュアルを参照してください。

言語関連マニュアルには「/製品インストールパス/」と書かれた箇所があります。
「/製品インストールパス/」は版数により異なります。
言語環境を設定するmodulefileで環境変数FJSVXTCLANGAに「/製品インストールパス/」を設定しています。
