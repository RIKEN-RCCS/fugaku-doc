2. 提供中のコンパイラ種別¶

スーパーコンピュータ「富岳」では、複数のコンパイラを利用することができます。 コンパイラの使用方法は次のページを参照ください。

  • 計算ノード向けバイナリを生成するコンパイラ

  • ログインノード向けバイナリを生成するコンパイラ

計算ノードで実行するプログラムは計算ノード向けバイナリを生成するコンパイラを使用してコンパイルします。

ログインノード上で計算ノードで実行するバイナリを生成する場合は、富士通コンパイラ、LLVM、GCCのクロスコンパイラを利用します。
