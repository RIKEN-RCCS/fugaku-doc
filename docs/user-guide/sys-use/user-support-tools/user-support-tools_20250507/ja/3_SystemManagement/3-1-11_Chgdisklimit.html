3.1.11. データ領域割当変更¶

　[名前]
　chgdisklimit - 課題のデータ領域の割り当てを変更します
　[書式]
　chgdisklimit [オプション]
　[説明]
　chgdisklimit コマンドは、課題に割り当て可能なディスク容量の上限値を超えない範囲で、各VOLUMEのディスク容量制限を
　変更できる。課題責任者とシステム管理者がこのコマンドを利用可能できる。それ以外の利用者は利用できない。
　課題に割り当て可能なディスク容量が少ない場合は、課題のディスク容量の拡張を申請するか、他のVOLUMEの容量制限を
　下げることで、割り当てが可能になる。
　[オプション]
　 -g GROUP_NAME
　 グループ名を指定する。スペースに続けての複数指定、アスタリスクを使用した前方／中間／後方一致指定はできない。
　 単一、完全一致指定のみ可能である。

　 -m VOLUME_NAME
　 先頭の２桁はVOLUMEを、後の２桁はMDT番号を表している。
　 【例】vol0401

注意

-mオプションで指定するVOLUMEは、chkdiskspaceコマンドを利用して各VOLUMEのFREE（空き容量）を確認し、割り当てる量よりも空きがあるVOLUMEを選んでください。

　 -B QUOTA_LIMIT_VALUE
　 クォータ制限値[単位:TiB]（ハードリミット）

　 --help
　 コマンドのヘルプ内容を表示する。
　[復帰値]
　正常時: 0
　異常時: 1
　[備考]
　なし。
　[使用例]
　【例1】vol0401に追加で割り当てを行う場合。
　以下の例では、課題に割り当て可能なディスク容量が10TiBあり5TiBの割り当て可能な残量がある。
