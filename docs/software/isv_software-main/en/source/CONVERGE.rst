CONVERGE - Flow to Start Using :red:`(Discontinuation of provision)`
====================================================================

The step to using CONVERGE on Fugaku are as follows.

#. License Agreement
#. Contacting User Information
#. Receive license file, set password, and request license registration
#. Notice of completion for software configuration
#. How to use

.. figure:: FIGURES/converge.png
   :width: 6.29921in
   :height: 3.28777in

   Fig. 1 Workflow of using CONVERGE

1. License Agreement
--------------------

A license agreement is required to use CONVERGE on Fugaku.
Please contact the following software vendor for details of the license agreement.

   **[Software Vendors]**

   IDAJ Co., LTD.

   **[Product Information]**

   https://www.idaj.co.jp/product/converge/

   **[Contacting e-mail address]**

   info@idaj.co.jp

2. Contacting User Information
------------------------------

After signing the license agreement,
please inform the licensee of the following information about all users who will use CONVERGE on Fugaku.

-  HPCI-ID
-  Project Number

The provided information will be used for the following purposes.

-  Setting up the software usage environment
-  Inquiry support

3. Receive license file, set password, and request license registration
-----------------------------------------------------------------------

After the user information is informed, the licensee will send the following files and information.

-  license file
-  Password Setting Instructions
-  Where to send the password-set license file

To properly use your CONVERGE license on Fugaku, please follow the instructions to set a password on your license file.

The license file with the password set will be registered on Fugaku, so please send it to the address specified by the licensee when the license file is sent.

4. Notice of completion for software configuration
--------------------------------------------------

After the software environment settings are completed,
the Help Desk will send a notice of completion for the software usage environment to the target user.

5. How to use
-------------

See below for details on use.

`https://www.fugaku.r-ccs.riken.jp/doc_root/en/user_guides/ConvergeGuide/usage_converge_en.pdf <https://www.fugaku.r-ccs.riken.jp/doc_root/en/user_guides/ConvergeGuide/usage_converge_en.pdf>`__
