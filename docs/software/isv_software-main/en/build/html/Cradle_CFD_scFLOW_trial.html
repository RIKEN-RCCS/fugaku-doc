Cradle CFD scFLOW - Flow to Start Trial Use (Discontinuation of provision)¶

The step to start a trial use of Cradle CFD scFLOW on Fugaku are as follows.

  1. Application for Trial Use

  2. Notification of Approval and Usage Period

  3. Start of trial use

  4. End of trial use

1. Application for Trial Use¶

  1. Access the request form (https://fugaku.zendesk.com/hc/en-us/requests/new) of Fugaku Support Site.

  2. Select the “Request form” pull-down menu.

  3. Select “Trial use of commercial application” from the “Inquiry type” menu.

  4. Input the “Project ID”, “Subject” (e.g. scFLOW trial use) and “User name” forms.

  5. Select “Cradle CFD scFLOW” from the “Application name” menu.

  6. Put the information of the users in the “Supplement” field (see below).

  7. Check the “Agreed to the precautions” box in the “Notice” section.

  8. Push the “Submit” button.

[Filling in the “Supplement” field]

Please put the information in the field as follows.

  • Local accounts of the users

    The listed accounts will be permitted to access the software during the period of the trial use. Please put the local accounts of all the users.

  • Purpose of use

    Describe the purpose of the trial use (e.g. checking of operation or performance)

  • Preferred start date

    The period of the trial use is two weeks. Put the preferred start date in yyyy/mm/dd format. Please consider the following points to determine the date.

    • It takes at least two weeks (i.e. ten working days) between the submission of the application and the start of the trial use.

    • Check the schedule of Fugaku operation because periods of the maintenance or the large scale job execution are included in that of the trial use. System schedule: https://www.fugaku.r-ccs.riken.jp/en/operation_schedule

    We might not be able to meet your request about the start date for some reasons. You can request some additional preferences about the period. We will take into account them to determine the period but might ask you to give us other possible ones.

    [ Example of requesting the start and end date ]:

    (Mandatory)
- First-preferred start date: yyyy/mm/dd and end date: yyyy/mm/dd

(Optional)
- second-preferred: yyyy/mm/dd, third-preferred: yyyy/mm/dd
- excluding the period from yyyy/mm/dd to yyyy/mm/dd
- no preference

  • End date of your project

    Put the end date of your project that applies the trial use in yyyy/mm/dd format.

  • Information of a representative

    Put the following information. If the affiliation or department are different among users, put all of them.

    User information
    Affiliation:
    Department:
    Address of the department:
    MSC Agreement No. (if you have a license of Cradle CFD scFLOW):

[Conditions of Use]

This trial use program is only for evaluating the performance and usability of Cradle CFD scFLOW to determine if you start the actual use of it on Fugaku. Before starting the trial use, you have to agree to the terms “scFLOW evaluation license terms” (see below).

In addition, your user information will be provided to the vendor in advance because you should purchase a normal license of Cradle CFD scFLOW for the actual use of it on Fugaku.

For this purpose, R-CCS will provide the vendor with the following information that is provided at the application.

Note that the trial use might not be approved by R-CCS or the vendor.

[Information to be provided]

  • Affiliation

  • Department

  • Address

  • MSC Agreement No.

  • Purpose of use

2. Notification of Approval and Usage Period¶

R-CCS and the vendor check the application and notify whether the trial use is approved or not. If approved, the usage period is also notified.

Please note that the usage period of your first choice is not always approved. If the period is not convenient for you, tell us that with the procedure on the notification.

3. Start of Trial Use¶

There is no notification of the start of the trial use. Note that you cannot use Cradle CFD scFLOW before the start. See below or the notification of the approval for details on use.

https://www.fugaku.r-ccs.riken.jp/doc_root/en/user_guides/scFlowGuide/usage_scflow_en.pdf

4. End of Trial Use¶

There is no notification of the end of the trial use. Note that you cannot use Cradle CFD scFLOW after the end. The result of jobs is not guaranteed that are running or waiting for run at the end of the trial use.
