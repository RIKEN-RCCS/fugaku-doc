Vampir - Application Information¶

Vampir is available for the login nodes and the pre/post environment.

1. Application method¶

If you wish to use Vampir, please submit your application via Zendesk.

  1. Access the request form (https://fugaku.zendesk.com/hc/en-us/requests/new) of Fugaku Support Site.

  2. At first, select the “Request form” pull-down menu.

  3. Select “Use of commercial application” from the “Inquiry type” menu.

  4. Input the “Project ID”, “Subject” and “User name” forms.

  5. Select “Vampir” from the “Application name” menu.

  6. Select “Application to start” from the “Request to start/stop use” menu.

  7. Check the “Agreeed to the precautions” box.

  8. Push the “Submit” button.

2. Note¶

We will determine whether or not to approve the application based on the following points of view.

  • This service is authorized under an academic license and cannot be used for commercial purposes. Applications may be rejected if the applicant’s affiliation is not an educational or research institution.
