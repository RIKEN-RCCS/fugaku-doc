Cradle CFD scFLOW - Flow to Start Using¶

The step to using Cradle CFD scFLOW on Fugaku are as follows.

  1. License Agreement

  2. Inquiry for license server information

  3. Contacting License Server Information

  4. License file registration and change request

  5. Notice of completion for software configuration

  6. How to use

1. License Agreement¶

A license agreement is required to use Cradle CFD scFLOW on Fugaku. Please contact the following software vendor for details of the license agreement.

[Software Vendors]

Hexagon Manufacturing Intelligence

[Product Information]

https://www.hexagonmi.com/mscsoftware/en

2. Inquiry for license server information¶

After signing the license agreement, please use the template below to contact the Help Desk for license server information.

As a rule, license server information does not change. When updating the expiration date of a license file, the same license server information should be used.

  1. Access to Fugaku Support site (https://fugaku.zendesk.com/hc/en-us) and press “Request”

  2. Select “Use of commercial application” from “Request type”

  3. Select “Project ID” from “Project ID for this request”

  4. Enter “License Server Information Inquiry” in “Subject”

  5. Select “Cradle CFD scFLOW” from “Application Name”

  6. Press “Submit”

The Help Desk will notify you of the license server information.

3. Contacting License Server Information¶

Please contact the licensee with the license server information notified by the Help Desk in the “Inquiry for License Server Information” section.

4. License file registration and change request¶

After receiving the license file from the licensee, please follow the procedure below to send the license file. Please be sure to include the HPCI-ID and project number of one representative and all members who will use the license. The license file will be registered and configured by the commercial software maintenance staff.

Please follow the same procedure when you change the user (addition or deletion) or license file for a registered license. Please be sure to include the HPCI-ID and project number of one representative designated at the time of registration. The license file does not need to be attached if there are no changes.

  1. Access to Fugaku Support site (https://fugaku.zendesk.com/hc/en-us) and press “Request”

  2. Select “Use of commercial application” from “Request type”

  3. Select “Project ID” from “Project ID for this request”

  4. Enter “License File Registration Request” or “License File Change Request” in “Subject”

  5. Select “Cradle CFD scFLOW” from “Application Name”

  6. Enter inquiry details in “Supplement” with reference to the following [Sample]

  7. Attach the license file if necessary

  8. Press “Submit”

  • Setting up the software usage environment

  • Inquiry support

5. Notice of completion for software configuration¶

After the software environment settings are completed, the Help Desk will send a notice of completion for the software usage environment to the target user.

See below for details on use.
