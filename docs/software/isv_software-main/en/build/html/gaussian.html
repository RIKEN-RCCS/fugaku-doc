Gaussian - Application Information¶

Gaussian is available on the compute nodes. You can use it in the single node execution but cannot in the multiple node execution. In addition, GaussView is available on the pre/post environment.

If you wish to use Gaussian, please submit your application via Fugaku Support Site. Note that you can submit applications for multiple users in one form by the method described below.

1. Application method¶

  1. Access to Fugaku Support site (https://fugaku.zendesk.com/hc/en-us) and press “Request”

  2. Select “Use of commercial application” from “Request type”

  3. Select “Project ID” from “Project ID for this request”

  4. Enter “Start of use (Gaussian)” or “Terminate of use (Gaussian)” in “Subject”

  5. Select “Gaussian” from “Application name”

  6. Enter the user id (local account for Fugaku) of each users to start/terminate of use Gauusian in “Supplement”

  7. Check “Agreeed to the precautions” box in “Notice” section

  8. Press “Submit” button

2. Note¶

2.1 Approval of the application¶

We will determine whether or not to approve the application based on the following points of view. If two or more users are specified in the application, only approved users can use Gaussian. If there are any unapproved users, we will inform the applicant of the account name of the unapproved users and the reason.

  • If the users are a non-resident based on the information provided at the time of application for an applicant’s Fugaku account, we need to submit a non-resident check form to abide by compliance. In such cases, additional information may be requested.

  • If the users are software developers competing with Gaussian, the use of Gaussian by such users may be rejected due to the terms of use offered by the software vendor.

2.2 Inquiries about Gaussian¶

When we receive inquiries about Gaussian on Fugaku, the inquiries and the related information may be shared among us (R-CCS), our partners and the software vendors to investigate them. We appreciate your understanding in advance.
