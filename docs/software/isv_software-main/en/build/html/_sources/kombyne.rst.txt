Kombyne
================================================================================

Kombyne is an in-situ visualization tool for executing visualization-related processing during the simulation runs. This is realized by instrumenting FORTRAN or C/C++ based simulation codes with the Kombyne API.  

Webpage: http://kombyne.net

1. How to use
-------------

Include and Library files for using Kombyne as well as some sample codes are installed on the following location. In addition, a PDF Manual (Kombyne.pdf) can be found in the "docs" folder.

/vol0004/apps/opt/kombyne

2. Eligibility for usage
------------------------

Fugaku account holders including users from commercial organizations in addition to the academic and research institutions. However, we ask to read carefully the license information found in the PDF Manual.
