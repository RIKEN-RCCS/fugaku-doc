COLMINA CAE Particle-based Casting Simulator - Flow to Start Trial Use :red:`(Discontinuation of provision)`
============================================================================================================

The step to start a trial use of COLMINA CAE Particle-based Casting Simulator on Fugaku are as follows.

#. Application for Trial Use
#. Notification of Approval and Usage Period
#. Receipt of license file
#. Start of trial use
#. End of trial use

.. image:: FIGURES/colmina_exaparticles_trial_eng.png
   :width: 4.50477in
   :height: 2.59587in

1. Application for Trial Use
-------------------------------

#. Access the request form(https://fugaku.zendesk.com/hc/en-us/requests/new) of Fugaku Support Site.
#. Select the "Request form" pull-down menu.
#. Select the related “Subject ID” from the “Project ID for this request” menu.
#. Select the “Trial use of commercial software” from the “Inquiry type” menu.
#. Input the “Subject” (e.g., COLMINA CAE Particle-based Casting Simulator trial use)
#. Select “COLMINA CAE Particle-based Casting Simulator” from the “Application name” menu.
#. Put the information of the users in the "Supplement" field (see below).
#. Check the “Agreed to the precautions” box in the “Notice” section.
#. Push the “Submit” button.

**[Filling in the "Supplement" field]**

Please put the information in the field as follows.

- **Local accounts of the users**

  The listed accounts will be permitted to access the software
  during the period of the trial use. Please put the local accounts of 
  all the users.

- **Preferred Start Date**

  The period of the trial use is two weeks. Put the preferred start
  date in yyyy/mm/dd format. Please consider the following points to
  determine the date.

  - It takes \ **at least twelve working days** \
    between the submission of the application and the start of the
    trial use.
  - Check the schedule of Fugaku operation because periods of
    the maintenance or the large scale job execution are included in
    that of the trial use.
    System schedule: \ https://www.fugaku.r-ccs.riken.jp/en/operation_schedule

  We might not be able to meet your request about the start date for
  some reasons. You can request some additional preferences about the
  period. We will take into account them to determine the period but
  might ask you to give us other possible ones.

  [ Example of requesting the start and end date ]::

   (Mandatory)
   - First-preferred start date: yyyy/mm/dd and end date: yyyy/mm/dd
   
   (Optional)
   - second-preferred: yyyy/mm/dd, third-preferred: yyyy/mm/dd
   - excluding the period from yyyy/mm/dd to yyyy/mm/dd
   - no preference

- **End Date of your Project**

  Put the end date of your project that applies the trial use in yyyy/mm/dd format.

- **Information of users**

  Put the following information. If the affiliation or department are
  different among users, put all of them.

  User information
    | Affiliation:
    | Department:
    | Address of the department:
    
- **Note**

  The period of the trial use cannot be extended but can be added if you submit an application again after the current period ends and the vendor approves it. When you submit an application again, provide us how many times did you applied the application in past, how long additional period do you need, and why do you need the additional period, as well as the information you provided at the first application. Please describe as much details of these information as possible, because the vendor use them to judge if the application is approved or not.

**[Conditions of Use]**

1. Purpose of use

   This trial use program is only for evaluating the performance and usability of COLMINA CAE Particle-based Casting Simulator to determine if you start the actual use of it on Fugaku.

2. Approval of trial use

   Note that the trial use might not be approved by R-CCS or the vendor.

3. The period of the trial use

   The period of the trial use is two weeks per an application.  Only after the current period ends, the additional application is possible. If the vendor approves it, you have another period of trial use.

2. Notification of Approval and Usage Period
----------------------------------------------

R-CCS and the vendor check the application and notify whether the trial use is
approved or not. If approved, the usage period is also notified.

Note that your first-preferred usage period is not necessarily approved. If the period is not convenient for you, tell us that according to the procedure on the notification.

3. Receipt of license file
--------------------------

The license file is going to be sent you by the software vender via R-CCS on the answer of the ticket of this application after the approval of the period for trial use. Please refer to the link in the fourth section for the configuration of the license. The software is available after the start of trial use.

4. Start of Trial Use
----------------------

There is no notification of the start of the trial use. Note that you
cannot use CONVERGE before the start. See below or the notification of
the approval for details on use.

https://www.fugaku.r-ccs.riken.jp/doc_root/en/user_guides/EXAPARTICLESGuide/usage_colmina_exaparticles_en.pdf

4. End of Trial Use
--------------------

There is no notification of the end of the trial use. Note that you
cannot use CONVERGE after the end. The result of jobs is not guaranteed
that are running or waiting for run at the end of the trial use. 
The license file you received can be deleted after the end of trial use.
