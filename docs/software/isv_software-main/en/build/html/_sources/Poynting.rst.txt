Poynting - Flow to Start Using :red:`(Discontinuation of provision)`
====================================================================

The step to using Poynting on Fugaku are as follows.

#. License Agreement
#. Contacting User Information
#. Receive license file
#. Notice of completion for software configuration
#. How to use

.. figure:: FIGURES/poynting.png
   :width: 6.29921in
   :height: 3.28777in

   Fig. 1. Workflow of using Poynting

1. License Agreement
--------------------

A license agreement is required to use Poynting on Fugaku.
Please contact the following software vendor for details of the license agreement.

   **[Software Vendors]**

   Fujitsu Limited

   **[Product Information]**

   https://www.fujitsu.com/jp/solutions/business-technology/tc/sol/poynting/

   **[Contacting e-mail address]**

   pyntsup@strad.ssg.fujitsu.com

2. Contacting User Information
------------------------------

After signing the license agreement,
please inform the licensee of the following information about all users who will use Poynting on Fugaku.

-  HPCI-ID
-  Project Number

The provided information will be used for the following purposes.

-  Setting up the software usage environment
-  Inquiry support

3. Receive license file
------------------------

The license file will be sent to you by the licensee.
Receipt of the license file may occur after receipt of the notice of completion for the software usage environment.

4. Notice of completion for software configuration
--------------------------------------------------

After the software environment settings are completed,
the Help Desk will send a notice of completion for the software usage environment to the target user.

5. How to use
-------------

See below for details on use.

`https://www.fugaku.r-ccs.riken.jp/doc_root/en/user_guides/PoyntingGuide/usage_poynting_en.pdf <https://www.fugaku.r-ccs.riken.jp/doc_root/en/user_guides/PoyntingGuide/usage_poynting_en.pdf>`__

.. Please start using Poynting according to the "Fugaku Commercial Software Poynting Usage Procedure" in the Fugaku User Portal.
