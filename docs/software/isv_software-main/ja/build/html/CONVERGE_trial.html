CONVERGE 試行利用開始までの流れ (提供を停止中)¶

「富岳」上に整備されたCONVERGEの試行利用を開始するために必要な手順は以下の通りです。

  1. 試行利用申請

  2. 利用可否・利用期間通知

  3. 合意書送付

  4. 試行利用開始

  5. 試行利用終了

1. 試行利用申請¶

  1. 富岳サポートサイトのお問い合わせ(https://fugaku.zendesk.com/hc/ja/requests/new) にアクセス

  2. 「各種申請」を選択

  3. 「お問い合わせ種別」から「商用アプリケーションの試行利用」を選択

  4. 「課題ID」、「件名」（例: [CONVERGE]試行利用申請）、「利用者氏名」を入力

  5. 「アプリケーション名」から「CONVERGE」を選択

  6. 「利用申請」に、利用申請者の情報を記入。詳細は以下「利用申請について」を参照してください

  7. 「利用条件・注意事項」を確認の上、同意チェックボックスにチェック

  8. 「送信」を押す

[利用申請について]

利用申請には、以下の内容を記入します。

  • 利用を希望するローカルアカウント

    連絡いただいたアカウントに対して、試行利用期間中に対象アプリケーションを実行可能とする設定を行います。
    利用を希望する利用者全員のローカルアカウントを記入してください。

  • 計算内容

    可能な範囲で計算内容を記載してください。記載内容、フォーマットなど指定はありません。

  • 試行利用希望開始日

    試行利用の期間は1週間です。試行利用の希望開始日を、yyyy/mm/dd 形式で記入してください。希望開始日は、以下を考慮の上決定してください。

    • 試行利用の申請から試行利用開始まで、最短で12営業日必要です

    • 利用開始日の2営業日前までに後述の合意書を署名・記入の上、提出いただきます。合意書の準備にかかる期間の考慮もお願いします

    • 利用不可となる保守期間や利用できるノード数が制限される大規模実行期間なども、試行利用の期間としてカウントされますので、運用スケジュールをご確認ください
      運用スケジュール：https://www.fugaku.r-ccs.riken.jp/operation_schedule

    なお、諸般の事情により試行利用の開始日を変更させていただく場合があります。試行利用期間について追加の希望があればご記入ください。可能な限り考慮しますが場合によっては別途日程調整を依頼する可能性がありますのでご了承ください。

    [ 試行利用希望開始日 記入例 ]:

    試行利用希望開始日（第一希望）：yyyy/mm/dd ※必須

追加希望（以下、記入例）
・第二希望日：yyyy/mm/dd 第三希望日 yyyy/mm/dd
・yyyy/mm/dd～yyyy/mm/dd の期間は除外してください
・いつでも大丈夫です

  • 課題終了日

    課題終了日（試行利用する課題の課題実施期間の終了日）を、yyyy/mm/dd 形式で記入してください。

  • 利用者情報（代表者1名）

    利用者の中から代表者1名について、以下の情報を記入してください。代表者は利用申請者以外でも問題ありません。本項で連絡いただいた代表者の方には、別途合意書も提出いただきます。

    利用者情報
    会社・組織名：
    部署名・役職：
    氏名：
    会社住所：※所属部署の事務所住所を記入してください。
    連絡先電話番号：
    連絡先メールアドレス：

  • その他

    利用期間の延長は受け付けておりませんが、利用期間終了後に再度試行利用申請を行い、ソフトウェアベンダーが特別に認めた場合に限り試行利用期間の追加が可能です。試行利用期間の追加を申請する場合、初回申請時に記入した内容に加え「過去に試行利用を利用した回数」「追加で必要な試行利用期間」「試行利用期間の追加が必要な理由」を記入してください。ソフトウェアベンダーの利用判定情報として使用されますので、可能な限り詳細に記載してください。

[利用条件・注意事項]

本試行利用はCONVERGEの性能や使い勝手の評価をして、その後の富岳の本格利用の実施可否を判断する目的にのみ利用いただけます。利用にあたって後述の合意書「7-DAY SOFTWARE EVALUATION AGREEMENT」への署名・ソフトウェアベンダーへの提出が必要です（合意書は利用日程の確定後提出いただきます）。

また、評価後に富岳の本格利用を実施する場合、CONVERGEの正式ライセンスを入手し富岳で利用することが前提となるため、あらかじめソフトウェアベンダーに利用者情報を提供する必要があります。

そのため、申請時に記入頂いた個人情報を含む以下の情報を、R-CCSが申請者の代理としてソフトウェアベンダーに提供します。

現在のところ、日本に居住する方が日本国内で利用する場合に限り、本試行利用を申請できます。

なお、R-CCSおよびソフトウェアベンダーの判断により利用が認められない場合がありますので、ご了承ください。

[提供情報]

  • 氏名

  • 会社・組織名

  • 部署名

  • 役職

  • 住所

  • 連絡先電話番号

  • 連絡先メールアドレス

  • 計算内容

2. 利用可否・利用期間通知¶

申請内容に対して、R-CCSおよびソフトウェアベンダーで試行利用が認められるか確認を行い、可否を通知します。試行利用が許可された場合、試行利用期間等を併せて通知します。

なお、第一希望の利用期間となるとは限りません。もし、通知された利用期間に問題があった場合などは通知文の手順に従い連絡してください。

利用期間の確定後、速やかに”3.合意書の提出”を実施してください。

3. 合意書の提出¶

試行利用期間の確定後、以下の提出方法に従いソフトウェア試行利用に関する合意書を提出してください。

ソフトウェアベンダーへ合意書を提出後に試行利用のための設定作業を実施しますので、速やかにご提出願います。

[提出方法]

  1. urlから合意書「7days CVG EVATUATION AGREEMENT for Fugaku via Riken.pdf」をダウンロードします。なお、合意書については予告なく内容を変更する場合がありますので、最新の合意書をご利用ください。

  2. 合意書の内容を確認し、必要事項を記入します。
    合意書の記入時は、以下の点にご注意ください。

  • 試行利用申請時に連絡いただいた代表者の情報を記入してください

  • 全て英語で記入してください

  • SIGNATURE 欄は直筆で署名してください

  • DATE欄には、利用開始日をyyyy/mm/dd 形式で記載してください。

  • PDF形式で保存してください

  1. 合意書については、利用可否・利用期間通知に添付のうえ返信にて提出してください。

4. 試行利用開始¶

試行利用開始の通知は行いません。試行利用期間前にアプリケーションは実行できませんのでご注意ください。利用方法については、利用可否・利用期間通知および下記リンク先を参照ください。

https://www.fugaku.r-ccs.riken.jp/doc_root/ja/user_guides/ConvergeGuide/usage_converge_ja.pdf

5. 試行利用終了¶

試行利用終了の通知は行いません。試行利用期間後にアプリケーションは実行できませんのでご注意ください。アプリケーション実行中やジョブの実行待ち中に試行利用期間を終了した場合、実行ジョブの保証はできません。
