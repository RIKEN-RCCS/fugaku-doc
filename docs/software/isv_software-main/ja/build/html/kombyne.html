Kombyne¶

シミュレーションと同時に可視化処理を行う「In-Situ可視化」向けツールです。FORTRAN や C/C++ のシミュレーション・コードに Kombyne APIを組込んでの利用となります。

Webページ：http://kombyne.net

1. 利用方法¶

組込みと実行向けに必要なライブラリやサンプル・コード等は以下にインストールされています。また、PDFドキュメント(Kombyne.pdf)は「docs」フォルダにあります。

/vol0004/apps/opt/kombyne

2. 利用資格¶

富岳アカウントを有する利用者であれば、教育機関、研究機関のみならず企業の方も利用できますが、利用前にライセンスの詳細をPDFドキュメントでご確認をお願いします。

/vol0004/apps/opt/kombyne/docs/Kombyne.pdf

3. ユーザー・サポート¶

質問や技術的サポート依頼は富岳サポートサイト（https://fugaku.zendesk.com/hc/ja/requests/new）よりお願いします。
