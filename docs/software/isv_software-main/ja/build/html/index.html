商用ソフトウェア¶

富岳では、以下の商用ソフトウェアが利用できます。
利用申請手続きについては、リンク先をご参照ください。

  • Amber

  • Ansys Fluent

  • Ansys LS-DYNA

  • AVS/Express

  • CONVERGE (提供を停止中)

  • Cradle CFD scFLOW

  • EXAMAG LLGシミュレータ

  • EXAPARTICLES/FLOW

  • Gaussian

  • Kombyne

  • MATLAB

  • Poynting (提供を停止中)

  • Simcenter STAR-CCM+

  • Vampir

  • VASP

また、EXAMAG LLGシミュレータ、EXAPARTICLES/FLOW、CONVERGE、Cradle CFD scFLOWについては、期限付きで試行的に利用できます。
試行利用の申請手続きについては、リンク先をご参照ください。

  • EXAMAG LLGシミュレータの試行利用 (提供を停止中)

  • EXAPARTICLES/FLOWの試行利用 (提供を停止中)

  • CONVERGEの試行利用 (提供を停止中)

  • Cradle CFD scFLOWの試行利用 (提供を停止中)

各ソフトウェアのバージョンについては、以下のリンク先をご参照ください。
