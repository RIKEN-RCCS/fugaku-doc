EXAPARTICLES/FLOW (旧名称: COLMINA CAE 粒子法鋳造解析) 利用開始までの流れ¶

「富岳」上に整備されたEXAPARTICLES/FLOWの利用を開始するために必要な手順は以下の通りです。

  1. ライセンス契約・利用者情報の連絡

  2. ライセンスファイルの受領

  3. ソフトウェア利用環境設定完了通知の受領

  4. 利用開始

1. ライセンス契約・利用者情報の連絡¶

「富岳」上に整備されたEXAPARTICLES/FLOWを利用するためには、ライセンス契約が必要です。ライセンス契約の詳細は下記ソフトウェアベンダーにお問い合わせください。

[提供ソフトウェアベンダー]

エフサステクノロジーズ株式会社

[製品情報サイト]

https://www.fsastech.com/products/software/exaparticles-flow/

ライセンス契約の締結後、「富岳」上に整備されたEXAPARTICLES/FLOWを利用する全ユーザーについて以下の情報をライセンス締結先にご連絡ください。

  • 富岳ローカルアカウント（未所持の場合、HPCI IDおよび課題番号）

なお、提供いただいた情報は以下の目的に使用します。

  • ソフトウェア利用環境の設定

  • お問合せ対応

2. ライセンスファイルの受領¶

ライセンス締結先よりライセンスファイルが送付されます。ライセンスファイルの受領はソフトウェア利用環境設定完了通知の受領後となる場合があります。

3. ソフトウェア利用環境設定完了通知の受領¶

ソフトウェア利用環境の設定完了後、富岳サポートより連絡対象ユーザー宛に「ソフトウェア利用環境設定完了通知」が送付されます。

4. 利用開始¶

利用方法については下記リンク先を参照ください。

https://www.fugaku.r-ccs.riken.jp/doc_root/ja/user_guides/EXAPARTICLESGuide/usage_colmina_exaparticles_ja.pdf
