Cradle CFD scFLOW 利用開始までの流れ¶

「富岳」上に整備されたCradle CFD scFLOWの利用を開始するために必要な手順は以下の通りです。

  1. ライセンス契約

  2. ライセンスサーバー情報の問合せ

  3. ライセンスサーバー情報の連絡

  4. ライセンスファイル登録・変更依頼

  5. ソフトウェア利用環境設定完了通知の受領

  6. 利用開始

1. ライセンス契約¶

「富岳」上に整備されたCradle CFD scFLOWを利用するためには、ライセンス契約が必要です。ライセンス契約の詳細は下記ソフトウェアベンダーにお問い合わせください。

[提供ソフトウェアベンダー]

Hexagon Manufacturing Intelligence社

[製品情報サイト]

https://www.hexagonmi.com/mscsoftware/jp

2. ライセンスサーバー情報の問合せ¶

ライセンス契約の締結後、以下のテンプレートを使用してライセンスサーバーの情報をヘルプデスクにお問い合わせください。

ライセンスサーバーの情報は原則として変更することはありません。ライセンスファイルの期限を更新する場合など、同じライセンスサーバー情報を使用してください。

  1. 富岳サポートサイト(https://fugaku.zendesk.com/hc/ja) にアクセスし、「申請する」を押す

  2. 「申請の種別」から、「商用アプリケーションの利用」を選択

  3. 「本件の対象となる課題ID」から、該当の課題IDを選択

  4. 「件名」に、「ライセンスサーバー情報 問合せ」と入力

  5. 「アプリケーション名」から、「Cradle CFD scFLOW」を選択

  6. 「送信」を押す

ヘルプデスクよりライセンスサーバー情報を通知します。

3. ライセンスサーバー情報の連絡¶

「ライセンスサーバー情報の問合せ」でヘルプデスクから通知されたライセンスサーバーの情報を、ライセンス締結先にご連絡ください。

4. ライセンスファイル登録・変更依頼¶

ライセンス締結先からライセンスファイルを受領後、以下の手順でライセンスファイルを送付してください。その際、必ず代表者1名とライセンスを利用する全員の富岳のローカルアカウントを記載してください。送付頂いたライセンスファイルは商用ソフトウェア保守担当によって登録・環境設定されます。

なお、登録済みのライセンスに関して利用者の変更（追加・削除）やライセンスファイルの変更などが発生した場合も、同様の手順で変更内容をお送りください。その際、必ず登録時に指定した代表者1名の富岳のローカルアカウントを記載してください。ライセンスファイルについては変更が無ければ添付は不要です。

  1. 富岳サポートサイト(https://fugaku.zendesk.com/hc/ja) にアクセスし、「申請する」を押す

  2. 「申請の種別」から、「商用アプリケーションの利用」を選択

  3. 「本件の対象となる課題ID」から、該当の課題IDを選択

  4. 「件名」に、「ライセンスファイル登録・変更依頼」と入力

  5. 「アプリケーション名」から、「Cradle CFD scFLOW」を選択

  6. 以下の[本文例]を参考に、「補足事項」を入力

  7. 必要に応じてライセンスファイルを添付する

  8. 「送信」を押す

5. ソフトウェア利用環境設定完了通知の受領¶

ライセンス登録などのソフトウェア利用環境の設定が完了後、ヘルプデスクより連絡対象利用者宛に「ソフトウェア利用環境設定登録完了通知」が送付されます。
