CONVERGE 利用開始までの流れ (提供を停止中)¶

「富岳」上に整備されたCONVERGEの利用を開始するために必要な手順は以下の通りです。

  1. ライセンス契約

  2. 利用者情報の連絡

  3. ライセンスファイルの受領、パスワード設定、ライセンス登録依頼

  4. ソフトウェア利用環境設定完了通知の受領

  5. 利用開始

1. ライセンス契約¶

「富岳」上に整備されたCONVERGEを利用するためには、ライセンス契約が必要です。ライセンス契約の詳細は下記ソフトウェアベンダーにお問い合わせください。

[提供ソフトウェアベンダー]

株式会社IDAJ

[製品情報サイト]

https://www.idaj.co.jp/product/converge/

[お問合せメールアドレス]

info@idaj.co.jp

2. 利用者情報の連絡¶

ライセンス契約の締結後、「富岳」上に整備されたCONVERGEを利用する全ユーザーについて以下の情報をライセンス締結先にご連絡ください。

  • HPCI-ID

  • 課題番号

なお、提供いただいた情報は以下の目的に使用します。

  • ソフトウェア利用環境の設定

  • お問合せ対応

3. ライセンスファイルの受領、パスワード設定、ライセンス登録依頼¶

利用者情報の連絡後、ライセンス締結先より以下のファイルおよび情報が送付されます。

  • ライセンスファイル

  • パスワード設定手順

  • パスワード設定済みライセンスファイルの送付先

「富岳」でCONVERGEのライセンスを適切に利用して頂くために、手順に従いライセンスファイルにパスワードを設定してください。

パスワード設定済みのライセンスファイルは、「富岳」環境でのライセンス登録を行いますので、ライセンス締結先よりライセンスファイル送付時に指定された送付先にお送りください。

4. ソフトウェア利用環境設定完了通知の受領¶

ソフトウェア利用環境の設定完了後、ヘルプデスクより連絡対象ユーザー宛に「ソフトウェア利用環境設定完了通知」が送付されます。

5. 利用開始¶

利用方法については下記リンク先を参照ください。

https://www.fugaku.r-ccs.riken.jp/doc_root/ja/user_guides/ConvergeGuide/usage_converge_ja.pdf
