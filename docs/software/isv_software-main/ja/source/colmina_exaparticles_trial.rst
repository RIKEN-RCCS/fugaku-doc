COLMINA CAE 粒子法鋳造解析 試行利用開始までの流れ :red:`(提供を停止中)`
=======================================================================

「富岳」上に整備されたCOLMINA CAE 粒子法鋳造解析の試行利用を開始するために必要な手順は以下の通りです。

#. 試行利用申請
#. 利用可否・利用期間通知
#. ライセンスファイル受領
#. 試行利用開始
#. 試行利用終了

.. image:: FIGURES/colmina_exaparticles_trial_jpn.png
   :width: 4.50477in
   :height: 2.59587in

1. 試行利用申請
---------------

#. 富岳サポートサイトのお問い合わせ(https://fugaku.zendesk.com/hc/ja/requests/new)にアクセス
#. 「各種申請」を選択
#. 「本件の対象となる課題ID」で該当する課題IDを選択
#. 「お問い合わせ種別」から「商用アプリケーションの試行利用」を選択
#. 「件名」（例:COLMINA CAE 粒子法鋳造解析試行利用申請）を入力
#. 「アプリケーション名」から「COLMINA CAE 粒子法鋳造解析」を選択
#. 「補足事項」に、利用申請者の情報を記入。詳細は以下「補足事項について」を参照してください
#. 「利用条件」を確認の上、同意チェックボックスにチェック
#. 「送信」を押す

**[「補足事項」について]**

「補足事項」には、以下の内容を記入します。

- **利用を希望するローカルアカウント**

  | 連絡いただいたアカウントに対して、試行利用期間中に対象アプリケーションを実行可能とする設定を行います。
  | 利用を希望する利用者全員のローカルアカウントを記入してください。

- **試行利用希望開始日**

  試行利用の期間は2週間です。試行利用の希望開始日を、yyyy/mm/dd 形式で記入してください。希望開始日は、以下を考慮の上決定してください。

  - 試行利用の申請から試行利用開始まで、\ **最短で12営業日必要**\ です
  - | 利用不可となる保守期間や利用できるノード数が制限される大規模実行期間なども試行利用の期間としてカウントされますので、運用スケジュールをご確認ください
    | 運用スケジュール：\ https://www.fugaku.r-ccs.riken.jp/operation_schedule

  なお、諸般の事情により試行利用の開始日を変更させていただく場合があります。試行利用期間について追加の希望があればご記入ください。可能な限り考慮しますが場合によっては別途日程調整を依頼する可能性がありますのでご了承ください。

  [ 試行利用希望開始日・課題終了日 記入例 ]::
   試行利用希望開始日（第一希望）：yyyy/mm/dd ※必須

   その他要望（以下、記入例）
   
   - 第二希望日：yyyy/mm/dd  第三希望日 yyyy/mm/dd
   - yyyy/mm/dd～yyyy/mm/dd の期間は除外してください
   - いつでも大丈夫です


- **課題終了日**

  本件の対象となる課題IDの課題終了日（当該課題の課題実施期間の終了日）を、yyyy/mm/dd 形式で記入してください。

- **利用者情報**

  以下の情報を記入してください。なお、利用者間で所属会社・組織または所属部署が異なる場合、全ての所属会社/部署の情報を提供してください。

   利用者情報
    | 会社・組織名：
    | 部署名：
    | 住所：※所属部署の事務所住所を記入してください。

- **その他**

  利用期間の延長は受け付けておりませんが、利用期間終了後に再度試行利用申請を行い、ソフトウェアベンダーが認めた場合に限り試行利用期間の追加が可能です。試行利用期間の追加を申請する場合、初回申請時に記入した内容に加え「過去に試行利用を利用した回数」「追加で必要な試行利用期間」「試行利用期間の追加が必要な理由」を記入してください。ソフトウェアベンダーの利用判定情報として使用されますので、可能な限り詳細に記載してください。

**[ 利用条件 ]**

1. 利用目的

   本試行利用はCOLMINA CAE 粒子法鋳造解析の性能や使い勝手の評価をして、その後の富岳の本格利用の実施可否を判断する目的にのみ利用いただけます。

2. 利用可否について

   R-CCSおよびソフトウェアベンダーの判断により利用が認められない場合がありますので、ご了承ください。

3. 試行利用期間

   1回の試行利用期間は2週間とします。利用期間終了後に再度試行利用申請を行い、ソフトウェアベンダーが認めた場合に限り試行利用期間の追加が可能です。

4. 利用者情報のご提供

   | 評価後に富岳の本格利用を実施する場合、COLMINA CAE 粒子法鋳造解析の正式ライセンスを入手し富岳で利用することが前提となるため、あらかじめソフトウェアベンダーに利用者情報を提供する必要があります。
   | そのため、申請時に記入頂いた以下の情報を、R-CCSが申請者の代理としてソフトウェアベンダーに提供します。

   **[提供情報]**

   • 会社・組織名
   • 部署名
   • 住所

2. 利用可否・利用期間通知
-------------------------

申請内容に対して、R-CCSおよびソフトウェアベンダーで試行利用が認められるか確認を行い、可否を通知します。
試行利用が許可された場合、試行利用期間を併せて通知します。

なお、第一希望の利用期間となるとは限りません。もし、通知された利用期間に問題などがあった場合は通知文の手順に従い連絡してください。

3. ライセンスファイル受領
-------------------------

利用期間が確定後、利用開始の前までに本申請のチケットへの回答にてソフトウェアベンダーから理化学研究所経由でライセンスファイルが送られてきます。下記４章のリンク先を参照してライセンスの設定してください。ソフトウェアは試行利用開始後に利用可能になります。

4. 試行利用開始
---------------

試行利用開始の通知は行いません。試行利用期間前にアプリケーションは実行できませんので、ご注意ください。利用方法については、利用可否・利用期間通知および下記リンク先を参照ください。

https://www.fugaku.r-ccs.riken.jp/doc_root/ja/user_guides/EXAPARTICLESGuide/usage_colmina_exaparticles_ja.pdf

5. 試行利用終了
---------------
試行利用終了の通知は行いません。試行利用期間後にアプリケーションは実行できませんのでご注意ください。アプリケーション実行中やジョブの実行待ち中に試行利用期間が終了した場合、実行ジョブの保証はできません。受領されたライセンスファイルは削除いただいて構いません。
