Poynting 利用開始までの流れ :red:`(提供を停止中)`
=================================================

「富岳」上に整備されたPoyntingの利用を開始するために必要な手順は以下の通りです。

#. ライセンス契約
#. 利用者情報の連絡
#. ライセンスファイルの受領
#. ソフトウェア利用環境設定完了通知の受領
#. 利用開始

.. figure:: FIGURES/poynting.png
   :width: 6.29921in
   :height: 3.28777in

   図1. Poynting利用開始までの作業フロー

1. ライセンス契約
-----------------

「富岳」上に整備されたPoyntingを利用するためには、ライセンス契約が必要です。ライセンス契約の詳細は下記ソフトウェアベンダーにお問い合わせください。

   **[提供ソフトウェアベンダー]**

   富士通株式会社

   **[製品情報サイト]**

   https://www.fujitsu.com/jp/solutions/business-technology/tc/sol/poynting/

   **[お問合せメールアドレス]**

   pyntsup@strad.ssg.fujitsu.com

2. 利用者情報の連絡
-------------------

ライセンス契約の締結後、「富岳」上に整備されたPoyntingを利用する全ユーザーについて以下の情報をライセンス締結先にご連絡ください。

-  HPCI-ID
-  課題番号

なお、提供いただいた情報は以下の目的に使用します。

-  ソフトウェア利用環境の設定
-  お問合せ対応

3. ライセンスファイルの受領
---------------------------

ライセンス締結先よりライセンスファイルが送付されます。ライセンスファイルの受領はソフトウェア利用環境設定完了通知の受領後となる場合があります。

4. ソフトウェア利用環境設定完了通知の受領
-----------------------------------------

ソフトウェア利用環境の設定完了後、ヘルプデスクより連絡対象ユーザー宛に「ソフトウェア利用環境設定完了通知」が送付されます。

5. 利用開始
-----------

利用方法については下記リンク先を参照ください。

`https://www.fugaku.r-ccs.riken.jp/doc_root/ja/user_guides/PoyntingGuide/usage_poynting_ja.pdf <https://www.fugaku.r-ccs.riken.jp/doc_root/ja/user_guides/PoyntingGuide/usage_poynting_ja.pdf>`__

.. 「富岳」利用者ポータル内の「富岳商用ソフトウェアPoynting利用手順書」に従いPoyntingの利用を開始してください。
